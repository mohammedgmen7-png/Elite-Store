# 📝 Todo List Application

تطبيق قائمة المهام (To-Do List) مع تخزين محلي

## ✨ المميزات

- ✅ إضافة وحذف وتحديث المهام
- ✅ تحديد المهام كـ مكتملة
- ✅ تصنيفات وفئات للمهام
- ✅ تاريخ استحقاق للمهام
- ✅ بحث وتصفية المهام
- ✅ تخزين محلي (Local Storage)
- ✅ واجهة مستخدم حديثة
- ✅ Responsive Design
- ✅ Dark Mode

## 🚀 البدء السريع

```bash
npm install
npm start
```
